之前说过，由于小程序对 npm 的支持和限制，所以能供小程序使用的包却 "为数不多".
这里注意三点：
1. 在项目根目录中右击，点击'外部终端窗口中打开', 在终端中下载 npm 包；
2. 点击工具，选择'构建 npm' 选项；
3. 构建 npm 前，应该先把根目录的miniprogram_npm 文件夹删除；
